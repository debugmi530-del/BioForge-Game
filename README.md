# BioForge 🦴

**3D конструктор существ с эволюционным ИИ — собери своё создание, запусти симуляцию и наблюдай как ИИ учит его ходить.**

---

## Геймплей

### Строитель
| Деталь | Описание |
|--------|----------|
| 🟠 Шарнир (оранжевый) | Шаровой сустав — кости двигаются свободно во всех направлениях |
| 🔵 Крепление (синее) | Жёсткое соединение — кости не вращаются относительно друг друга |
| ⬛ Кость | Чёрный жёсткий цилиндр — соединяет два шарнира/крепления |
| 🔴 Мышца | Красный элемент (-<>-) — прикрепляется к костям, тянет/толкает их |

**Правила крепления:**
- Кости → только к шарнирам или креплениям
- Шарниры/крепления → только к костям
- Мышцы → только к костям

**Симметрия:** инструмент симметрии (2–10) зеркалит размещение деталей по оси Y

### Симуляция ИИ
- **Эволюционный алгоритм** подбирает параметры мышц (сила, частота, фаза)
- Каждое поколение мутирует лучший геном на ±0.001
- Функция приспособленности = расстояние × 0.5 + скорость × 0.3 + стабильность × 0.2
- Популяция: от 1 до 1 000 000 особей
- Режимы: Авто / Бег / Ходьба / Прыжки / Всё вместе

## Установка и запуск

### Требования
- [Godot 4.2+](https://godotengine.org/download)

### Запуск в редакторе
1. Клонируй репозиторий: `git clone https://github.com/debugmi530-del/BioForge-Game.git`
2. Открой Godot 4 → **Import** → выбери папку `godot_creature_game/`
3. Нажми **Play (F5)**

### Windows инсталлятор
Скачай последний `BioForge-Setup.exe` из [Releases](https://github.com/debugmi530-del/BioForge-Game/releases).  
Сборка происходит автоматически через GitHub Actions при каждом пуше в `main` или новом теге `v*`.

## Структура проекта

```
godot_creature_game/
├── scripts/
│   ├── autoload/          # GameManager, CreatureData (глобальные)
│   ├── builder/           # Строитель: части, логика размещения
│   │   └── parts/         # BonePart, MusclePart, JointPoint, ConnectionPoint
│   ├── simulation/        # Физика, ИИ, SimulationManager
│   ├── camera/            # Орбитальная камера
│   └── ui/                # Все экраны UI
├── scenes/
│   ├── main_menu.tscn
│   ├── builder.tscn
│   ├── simulation.tscn
│   └── parts/             # Мини-сцены для каждой детали
├── assets/                # Иконка и ресурсы
└── .github/workflows/     # CI/CD сборка инсталлятора
```

## CI/CD — автосборка

GitHub Actions запускается при:
- `push` в ветку `main`
- Теге вида `v1.0.0`
- Ручном запуске

Результат: `BioForge-Setup.exe` (Windows x64 инсталлятор с ярлыком на рабочем столе).
